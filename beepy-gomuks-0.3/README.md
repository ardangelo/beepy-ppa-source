# Beepy Gomuks

## Setting up Gomuks

```
     ╔═══════════Log in to Matrix═══════════╗     
     ║                                      ║     
     ║ Email example@example.com            ║     
     ║                                      ║     
     ║       [Enter email to request code ] ║     
     ║                                      ║     
     ║ Code  123456                         ║     
     ║                                      ║     
     ║ [       Enter code to log in       ] ║     
     ║                                      ║     
     ║ [               Quit               ] ║     
     ╚══════════════════════════════════════╝
```
